# SchemeSathi — AI Government Scheme Assistant

MERN starter for a hackathon project that helps people find Indian government schemes.

## Run locally

1. In `server`, copy `.env.example` to `.env`, then run `npm run dev`.
2. In `client`, run `npm run dev`.

The demo uses a starter in-memory scheme catalogue. MongoDB is included as a dependency for the next step: persisting verified schemes and user profiles.
